# Task 3 - Attributes

## Images - Basic Image
!!! abstract "Instructions"
    **Using VS Code**, create a new `index.html` page and add an image to the webpage using the `img` element.

    Use:

    - `https://cyberbilby.com/capybara.jpg` as the image source
    - `Capybara` as the alt text

??? question "Hint"
    The image element uses the `img` tag. Images use the `src` attribute for the file location and the `alt` attribute for a description of the image.

## Images - Multiple Images

!!! abstract "Instructions"
    **Using VS Code**, create a new `gallery.html` page and add:

    - an image with the source `mountain.jpg`
    - an image with the source `forest.jpg`

    Add appropriate alt text for both images.

??? question "Hint"
    The `alt` attribute should describe the image content in a short sentence.

## External Links

!!! abstract "Instructions"
    **Using VS Code**, create a new `index.html` page and create a **link** that takes the user to `https://w3.org/` when clicked.

??? question "Hint"
    The tag name for the **link** element is called `a` and the **attribute** for links is `href`. Remember elements need an opening and closing tag.

## Internal Links

!!! abstract "Instructions"
    **Using VS Code**, create `index.html` and `about.html` pages. Create a **link** on both pages that allows the user to navigate between the two pages.

??? question "Hint"
    The tag name for the **link** element is called `a` and the **attribute** for links is `href`. Remember elements need an opening and closing tag.
