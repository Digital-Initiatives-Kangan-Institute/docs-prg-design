1. Introduction to HTML

This site will guide you through the fundamentals of how websites are built. You’ll learn how to create simple webpages, modify them by editing and adding new elements, and then link them together into a complete website.

1.1 What is HTML?

HyperText Markup Language (HTML) is the standard language used to create and structure webpages. HTML tells a web browser what each part of a page is, such as headings, paragraphs, images, and links. It uses simple tags to organise content so that it can be displayed correctly on the screen. Without HTML, a webpage would not have any structure or meaning.

<!DOCTYPE html>
<html>
  <head>
    <title>Example Web Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is a paragraph on my webpage.</p>

    <a href="https://w3.org/">Visit W3C</a>
  </body>
</html>

The above is a simple webpage that has a level 1 heading (h1), a paragraph (p) and a link (a) to the World Wide Web Consortium (W3C) website.
