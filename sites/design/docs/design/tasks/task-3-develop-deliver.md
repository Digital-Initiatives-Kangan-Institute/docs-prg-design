# Task 3 - Developer and Deliver a Solution
!!! warning "Important"
    This task must be done in groups of three.

## Instructions
You are required to create a storage device for marbles.

**Requirements for the storage are:**

- Must be 3D Printable (FDM)
- Store between 10-15 marbles
- Easy to insert and remove marbles
- Strong enough to hold the weight of marbles without breaking
- Should not tip over easily when placed on a surface
- No sharp edges
- Use a minimal amount of material

**Process**

1. Brainstorm and come up with some designs (Use a digital tool)
2. Choose a design and create a prototype
3. Print and test a prototype (Document findings in template provided)
4. Get feedback and repeat process to fix shortcomings (Use template provided)
